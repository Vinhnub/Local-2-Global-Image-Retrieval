# 🔄 Offline Modules Directory

## 📌 What it does
Contains scripts used to Digitize and Build the Index for the Database. 
This process is extremely time-consuming, but **ONLY NEEDS TO BE RUN ONCE**.

## 🗂 3 Stages:
1. `stage1_local_extract`: Extract local features (FIRe).
2. `stage2_global_extract`: Extract global features (CVNet).
3. `stage3_build_index`: Process data into the Dictionary (ASMK).
